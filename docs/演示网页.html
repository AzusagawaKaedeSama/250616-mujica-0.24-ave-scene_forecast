🎯 不确定性量化机制

动态不确定性计算公式

最终不确定性 = 基础不确定性 × 场景不确定性倍数 × 时段调整系数 × 置信度调整

不确定性来源

  • 数据不确定性：测量误差、数据缺失
  • 模型不确定性：参数估计误差
  • 场景不确定性：天气预报误差
  • 系统不确定性：负荷模式变化

时段调整系数

  • 早晨(6:00-9:00)：1.2x
  • 中午(11:00-14:00)：0.9x
  • 晚高峰(18:00-21:00)：1.3x
  • 夜间(22:00-5:00)：1.1x

🛠️ 系统部署与使用

🚀 快速开始

# 1. 负荷预测
python scripts/scene_forecasting.py --forecast_type=load --province=上海 --forecast_date=2024-06-10

python scripts/scene_forecasting.py --train_prediction_type=interval --confidence_level=0.95

📋 输入输出规范

📥 输入数据

  • 历史负荷数据：CSV格式，15分钟间隔
  • 天气数据：温度、湿度、风速、降水、辐射
  • 时间参数：预测日期、类型、置信水平

📤 输出结果

  • 点预测值：未来24小时逐15分钟预测(MW)
  • 区间预测：上下边界、区间宽度
  • 概率预测：多分位数、概率密度函数

⚠️ 输入不确定性来源

天气预报误差
温度±2-3°C，降水±20-30%，风速±1-2m/s
历史数据误差
负荷计量±1-2%，数据缺失、异常值
模型参数误差
训练样本局限性，泛化能力限制
